# Spark ETL Project

## Description
A Spark-based ETL pipeline for extracting, transforming, and loading data into Oracle Database.

## Project Structure
- **data/**: Contains mock data and database credentials.
- **src/**: Includes Python modules for table creation, data population, and ETL.
- **config/**: Spark session configuration.
- **logs/**: Logs for tracking ETL runs.
